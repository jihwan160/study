제이쿼리 효과

제이쿼리에서 화면전환, 슬라이드 다운과 업효과 등
다양한 효과 메서드를 제공
단, 제이쿼리 코어파일 버전에 따라 동작이 안될 경우도 있음

show / hide / toggle 효과

DOM선택 요소를 숨기거나 노출하고 싶을 경우에 사용
형식 : show('시간') | hide('시간') | toggle('시간')
매개변수 '시간' = 'fast' 200, 'normal' 400, 'slow' 600, 밀리세컨으로 입력 가능

slideDown / slideUp / slideToggle 효과

slideUp은 DOM선택요소를 위로 미끄러지듯 말아 올리면서 숨김
slideDown은 DOM선택요소를 아로로 미끄러지듯 말아 내리면서 노출시킴
형식 : show('시간') | hide('시간') | toggle('시간')
매개변수 '시간' = 'fast' 200, 'normal' 400, 'slow' 600, 밀리세컨으로 입력 가능

FadeIn / FadeOut / fateToggle

fadeOut은 DOM선택요소를 천천히 사라지게 함
fadeIn은 DOM선택요소를 천천히 나타나게 함
형식 : show('시간') | hide('시간') | toggle('시간')
매개변수 '시간' = 'fast' 200, 'normal' 400, 'slow' 600, 밀리세컨으로 입력 가능
